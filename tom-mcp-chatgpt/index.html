Local MCP workspace for ChatGPT Web

Cho ChatGPT Web đôi tay local, nhưng vẫn có rào an toàn.

Local MCP for ChatGPT tạo một MCP-style server chạy trên máy anh, giới hạn trong một workspace đã chọn. ChatGPT hoặc MCP client có thể đọc file, sửa file, tìm kiếm và chạy lệnh có approval.

Tính năng

Không phải API chui. Đây là bridge local có kiểm soát.

App này không bypass ChatGPT login hay Cloudflare. Nó chỉ cung cấp công cụ local cho một workspace an toàn.

Workspace sandboxChỉ cho phép thao tác trong thư mục anh chọn. Không nên trỏ vào toàn bộ ổ đĩa.
File + search toolsList/read/write/replace/search file để ChatGPT hiểu và chỉnh project local.
Command approvalChạy test/build với policy và approval gate cho lệnh rủi ro.
Security model

Dùng được, nhưng phải dùng đúng cách.

Public preview này dành cho demo folder trước. Không đưa folder nhạy cảm vào nếu chưa hiểu threat model.

Không lưu mật khẩuApp không lưu password/cookie ChatGPT và không phải unofficial ChatGPT Web API.
Local-firstServer mặc định bind local. Không expose internet nếu chưa có authentication/tunnel rõ ràng.
Audit trailApproval record và audit log được lưu trong workspace để xem lại tool đã làm gì.
Lưu ý: Bản public-safe nên dùng repo CD-Local-Agent-Studio. Repo private/internal không đưa lên web public.
Cách dùng

Chọn workspace, start server, copy MCP URL.

Sau đó dùng prompt hướng dẫn ChatGPT/MCP client thao tác trong workspace đó.

01 — Chọn folder demoBắt đầu bằng một thư mục an toàn, không chứa dữ liệu nhạy cảm.
02 — Start local serverGiữ mode safe + policy balanced, kiểm tra health endpoint local.
03 — Kết nối ChatGPTCopy MCP URL hoặc setup connector/tunnel phù hợp với môi trường ChatGPT của anh.
